Apa itu HTML ?


HTML (Hypertext Markup Language) merupakan gabungan dari dua istilah hypertext dan markup language.

Hypertext yaitu dokumen berisi tautan yang memungkinkan pengguna terhubung ke halaman lain. Sedangkan Markup Language merupakan bahasa komputer yang terdiri dari sekumpulan kode untuk mengatur struktur dan menyajikan informasi.

Jadi, HTML adalah bahasa markup untuk membuat struktur halaman website.


Apa itu Tag, Element, dan Attribute ?


  • Tag, digunakan sebagai awalan intruksi atau perintah yang akan dibaca web browser. Misalnya tag <strong> dibawah ini :

    <p>tag ini akan membuat tulisan menjadi<strong>tebal</strong></p>

    Tag ini akan meminta browser menampilkan teks dalam format tebal.

  • Element, , Merupakan keseluruhan kode yang terdiri dari tag pembuka <> hingga tag penutup </> . Contohnya :

    <h1>Contoh Element</h1>

    dan tidak semua element harus diakhiri dengan tag penutupnya seperti :

    <img scr="..." alt="...">

  • Attribute, yaitu informasi atau perintah tambahan yang berada didalam elemen. Misalnya attr src dan alt dibawah ini :

    src="..."

    alt="..."

    src bisa diartikan sebagai source atau sumber dari sebuah file atau gambar. alt bisa diartikan sebagai alternative description yang digunakan untuk memberi keterangan dari tag <img> yang gagal ditampilkan pada browser seperti berikut ini :
